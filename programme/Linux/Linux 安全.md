# Linux 安全

## 更新操作系统及软件

更新操作系统及软件能够很好的修复系统漏洞及软件漏洞。

```shell
sudo apt update && sudo apt upgrade -y
```

## 创建特权用户

- 创建新的账户拥有`sudo`用户
- 禁止`root`登录

## 禁止密码登录

- 禁止使用密码登录，而使用秘钥登录

`vim /etc/ssh/sshd_config`

PasswordAuthentication no

重启服务: `systemctl restart sshd.service`
